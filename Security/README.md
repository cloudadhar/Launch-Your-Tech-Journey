# Security Learning Path

## Foundation

- CIA triad
- Authentication and authorization
- IAM and RBAC
- Encryption at rest and in transit
- Secrets management
- Network segmentation
- Vulnerability management
- Threat modeling
- Logging and audit trails
- Incident response

## Cloud Security

- Least privilege IAM
- MFA and break-glass access
- KMS and key rotation
- Private networking
- Security groups and firewall rules
- Cloud audit logs
- Guardrails and policy as code
- Backup and disaster recovery

## Application Security

- OWASP Top 10
- Secure coding
- Input validation
- Dependency scanning
- SAST and DAST
- API security
- Container security
- API gateway authentication, authorization, rate limiting, and audit logs

## Practice Deliverables

- Threat model for a sample app
- Secure CI/CD pipeline
- IAM least privilege policy
- Container scan report
- Incident response report
- Secure Nginx or Kong gateway configuration review
