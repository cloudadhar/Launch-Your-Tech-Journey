# AWS Learning Path

## Order to Learn

1. Account safety, IAM users, roles, policies, MFA, billing alerts
2. Compute: EC2, AMI, Auto Scaling, Elastic Load Balancing
3. Storage: S3, EBS, EFS, lifecycle policies
4. Networking: VPC, subnets, route tables, NAT, security groups, NACLs
5. Databases: RDS, DynamoDB, ElastiCache
6. Containers: ECR, ECS, EKS
7. Serverless: Lambda, API Gateway, EventBridge
8. DevOps: CodePipeline, CodeBuild, CodeDeploy, CloudFormation
9. Observability: CloudWatch, X-Ray, CloudTrail
10. AI/ML: SageMaker AI, Bedrock, vector search patterns
11. Security: KMS, Secrets Manager, GuardDuty, Security Hub, IAM Access Analyzer
12. Cost: Budgets, Cost Explorer, tagging, cleanup automation

## Hands-on Labs

- Launch an EC2 web server and clean it up.
- Host a static website on S3 and CloudFront.
- Build a VPC with public and private subnets.
- Deploy Docker image to ECS or EKS.
- Build a CI/CD pipeline with GitHub Actions and Terraform.
- Monitor an app with CloudWatch alarms.
- Deploy a simple ML inference API.
