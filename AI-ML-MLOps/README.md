# AI / ML / MLOps Guide

## Learning Order

1. Python basics
2. NumPy, Pandas, Matplotlib
3. Statistics and probability
4. Machine learning fundamentals
5. Model evaluation
6. Feature engineering
7. Deep learning basics
8. Generative AI and LLM fundamentals
9. RAG and vector databases
10. Model serving APIs
11. Docker for model packaging
12. CI/CD for models
13. Model registry and experiment tracking
14. Monitoring, drift, and governance
