# Kubernetes Guide

## Order to Learn

1. Cluster, node, pod, deployment, service
2. ConfigMap and Secret
3. Ingress
4. Persistent volumes
5. Namespaces and RBAC
6. Health probes and resource requests
7. Helm charts
8. Autoscaling
9. Network policies
10. Observability
11. Security scanning and policy
12. GitOps with Argo CD

- Deploy Nginx.
- Deploy a web app with service and ingress.
- Add readiness and liveness probes.
- Package with Helm.
- Add Prometheus metrics.
- Deploy with Argo CD.
- Add network policy and resource limits.
