# Foundation Roadmap for Any Tech Path

Before choosing any domain, build your foundation.

## Core Skills

1. **Programming**
   - Start with Python, Java, C, or JavaScript
   - Learn variables, loops, functions, arrays, objects

2. **Problem Solving**
   - Practice small coding problems
   - Learn basic data structures

3. **Computer Fundamentals**
   - Operating systems
   - Networking basics
   - Databases
   - How web applications work

4. **Git and GitHub**
   - Create GitHub account
   - Push your projects
   - Learn commit, branch, pull request basics

5. **Linux Basics**
   - Files and folders
   - Permissions
   - Basic commands
   - Package installation

6. **Database Basics**
   - SQL basics
   - Tables, rows, columns
   - CRUD operations

7. **Communication Skills**
   - Explain your project clearly
   - Write README files
   - Practice interview answers

## First milestone

Build and upload one project to GitHub.
