# Cloud & DevOps Roadmap

## What this path means
Cloud & DevOps engineers run applications on cloud and automate deployment, infrastructure, and operations.

## Learning sequence

1. Linux basics
2. Networking basics
3. Git and GitHub
4. Cloud fundamentals: AWS / Azure / GCP
5. IAM and security basics
6. Compute, storage, database, networking services
7. CI/CD pipelines
8. Docker
9. Kubernetes
10. Infrastructure as Code with Terraform
11. Monitoring and logging
12. Cost awareness and reliability basics

## Projects

- Host a static website on cloud
- Deploy EC2 using Terraform
- CI/CD pipeline using GitHub Actions
- Dockerize a web application
- Kubernetes mini deployment
- Monitoring dashboard demo
