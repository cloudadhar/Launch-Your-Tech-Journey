# Google Cloud Learning Path

## Order to Learn

1. Organization, folders, projects, billing
2. IAM, service accounts, workload identity
3. Compute Engine, Cloud Run, Cloud Functions
4. Cloud Storage, persistent disks
5. VPC, firewall rules, load balancing, Cloud DNS
6. Cloud SQL, Firestore, BigQuery
7. Artifact Registry, GKE, Cloud Deploy
8. Cloud Build and CI/CD
9. Cloud Monitoring, Cloud Logging, Error Reporting
10. Vertex AI, Gemini, agent platforms, model deployment
11. Security Command Center, Secret Manager, KMS

## Hands-on Labs

- Deploy a static app to Cloud Run.
- Build and push an image to Artifact Registry.
- Deploy a GKE workload.
- Create a Cloud Build pipeline.
- Create logs, metrics, and alerts.
- Deploy a simple AI/ML API using managed services.
