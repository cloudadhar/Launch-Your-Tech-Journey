# Certification Guidance

## Important message

First build basics, then choose certification.

Certifications are useful when you already understand the fundamentals and have hands-on projects.

## Cloud Certifications

### Beginner
- AWS Certified Cloud Practitioner
- Microsoft Certified: Azure Fundamentals
- Google Cloud Digital Leader

### Associate / Intermediate
- AWS Certified Solutions Architect – Associate
- Microsoft Azure Administrator Associate
- Google Associate Cloud Engineer

### Advanced / Specialty
- AWS Security Specialty
- Google Professional Cloud Architect
- Google Professional Machine Learning Engineer

## DevOps / Cloud Native Certifications

- Kubernetes and Cloud Native Associate (KCNA)
- Certified Kubernetes Administrator (CKA)
- Certified Kubernetes Application Developer (CKAD)
- Certified Kubernetes Security Specialist (CKS)
- Terraform Associate

## AI / ML Certifications

- Azure AI Fundamentals
- Google Professional Machine Learning Engineer
- Cloud provider AI/ML learning paths
- Python, ML, and GenAI project-based courses

## Cyber Security Certifications

- Security fundamentals
- Cloud security fundamentals
- Ethical hacking basics
- Security operations basics

## How to choose

1. Choose your career path first
2. Build 2-3 projects
3. Learn fundamentals
4. Pick one certification
5. Prepare with labs, not only videos
