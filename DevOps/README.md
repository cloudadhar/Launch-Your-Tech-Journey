# DevOps Guide

## Core Concepts

- Source control
- Branching strategies
- Build automation
- Test automation
- Continuous integration
- Continuous delivery and deployment
- Infrastructure as Code
- Configuration management
- Containers and orchestration
- Observability
- Incident response
- Reliability engineering
- Cost and FinOps basics

## Learning Order

1. Linux and shell
2. Git and GitHub
3. Networking and HTTP
4. Scripting with Bash and Python
5. CI/CD with GitHub Actions and GitLab CI/CD
6. Docker
7. Kubernetes
8. Terraform
9. Ansible
10. Monitoring and logging
11. Security gates
12. Production troubleshooting

## Tools to Learn

- Git, GitHub, GitLab
- GitHub Actions, GitLab CI/CD, Jenkins
- Docker, Kubernetes, Helm
- Nginx, Kong Gateway
- Terraform, Ansible
- Prometheus, Grafana, OpenTelemetry
- SonarQube, Trivy, Checkov
- Nexus Repository, JFrog Artifactory
- Jira, Confluence, draw.io
