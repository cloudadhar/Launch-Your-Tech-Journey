# Technology Stack Maps

This folder explains how tools fit together across DevOps, cloud, full-stack, AI/ML, MLOps, and agentic AI. Use it when a learner asks, "Which tools should I learn together?"

## Stack Guides

- [DevOps Tool Stack](devops-tool-stack.md)
- [Cloud Tool Stack](cloud-tool-stack.md)
- [AI / ML / MLOps Tool Stack](ai-ml-mlops-tool-stack.md)
- [Full-Stack Developer Tool Stack](full-stack-tool-stack.md)

## How to Use These Guides

1. Learn the purpose of each layer.
2. Pick one beginner tool per layer.
3. Build one small project using the selected stack.
4. Document commands, errors, fixes, and diagrams.
5. Replace beginner tools with production-grade alternatives as you advance.
