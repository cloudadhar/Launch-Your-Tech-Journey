# Azure Learning Path

## Order to Learn

1. Tenant, subscription, resource group, region
2. Microsoft Entra ID, RBAC, managed identities
3. Compute: Virtual Machines, App Service, Functions
4. Storage: Blob Storage, Files, Disks
5. Networking: VNet, subnets, NSG, Load Balancer, Application Gateway
6. Databases: Azure SQL, Cosmos DB, PostgreSQL
7. Containers: ACR, AKS, Container Apps
8. DevOps: Azure DevOps, Boards, Repos, Pipelines, Artifacts
9. Observability: Azure Monitor, Log Analytics, Managed Grafana
10. AI/ML: Azure Machine Learning, Azure AI services
11. Security: Key Vault, Defender for Cloud, Sentinel

## Hands-on Labs

- Deploy a VM and expose a simple website.
- Build an AKS deployment from a container image.
- Create an Azure DevOps pipeline.
- Store app secrets in Key Vault.
- Create logs and dashboards with Azure Monitor and Grafana.
