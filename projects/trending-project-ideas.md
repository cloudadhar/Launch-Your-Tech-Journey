# Trending Project Ideas

## 1. AI College Assistant
A chatbot that answers questions about timetable, syllabus, assignments, college FAQs, and events.

### Skills used
Full-Stack, AI, RAG, APIs, database, deployment

## 2. Smart Assignment Reminder
An AI-powered reminder app for assignments, exams, daily classes, and deadlines.

### Skills used
Full-Stack, notifications, database, automation

## 3. AI Notes Maker
Upload class notes or text and generate clean PDF notes, summaries, and key points.

### Skills used
AI, PDF generation, Hugging Face / LLM APIs, backend, frontend

## 4. Class Point Taker
Convert lecture audio or rough notes into structured bullet points and revision notes.

### Skills used
Speech-to-text, AI summarization, PDF export, storage

## 5. Online Shopping Mini Platform
Product listing, search, cart, login, payment flow simulation, and admin dashboard.

### Skills used
Full-Stack, database, APIs, authentication, deployment

## 6. Cloud-Hosted Portfolio Website
A personal portfolio hosted on cloud and deployed automatically from GitHub.

### Skills used
HTML/CSS/JS, GitHub, cloud, CI/CD

## 7. CI/CD Deployment Project
Push code to GitHub and automatically deploy the application.

### Skills used
GitHub Actions, DevOps, cloud, deployment automation

## 8. AI Resume & Interview Helper
Generate resume suggestions, interview questions, and domain-based career guidance.

### Skills used
GenAI, prompt engineering, full-stack, PDF generation

## How to present projects in interviews

For every project, explain:

1. Problem statement
2. Technologies used
3. Architecture
4. Demo link
5. GitHub link
6. What you learned
7. What you will improve next
